BÍBLIA ACF IA GRÁTIS

Esse pacote funciona sem OpenAI e sem pagar API.

Estrutura:
- index.html
- data/acf_biblia_ptbr.json

Recursos:
- leitura por livro e capítulo
- busca
- favoritos locais
- anotações locais
- plano de leitura
- IA local grátis:
  - chat bíblico
  - explicação
  - reflexão
  - oração
  - estudo

Como usar:
1. descompacte
2. suba no GitHub mantendo a pasta data
3. faça deploy no Vercel
